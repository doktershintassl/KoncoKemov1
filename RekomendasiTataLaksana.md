# Pedoman Tata Laksana Gejala (Symptom Management Guidelines)

| Gejala | Tata Laksana (Tx) |
| :--- | :--- |
| **Mual / Muntah** | Ondansetron 8mg |
| **Diare 1-6x** | Absorbent |
| **Diare >6x** | Loperamide |
| **Nyeri 1-3** | Paracetamol |
| **Nyeri 4-6** | NSAID / Tramadol |
| **Nyeri 7-10** | Codein |
| **Sariawan** | Nystatin Oral Drops |
| **Dehidrasi** | Rehidrasi sesuai tingkat Dehidrasi |
| **Sesak Napas** | Terapi Oksigen sesuai saturasi |
| **Anemia** | Evaluasi kadar Hb; Lakukan transfusi sesuai indikasi berdasarkan rekomendasi Dokter Sp.PD atau KHOM setempat. |
| **Leukopenia** | Evaluasi hitung leukosit; Berikan terapi G-CSF sesuai indikasi berdasarkan rekomendasi Dokter Sp.PD atau KHOM setempat. |
| **Tanda Bahaya / Darurat** | Tata laksana awal sesuai indikasi untuk stabilisasi pasien, kemudian segera lakukan konsultasi/rujukan ke Dokter Sp.PD atau KHOM setempat. |
